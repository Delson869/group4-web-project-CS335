# Group 4 – Class Management Web Application (CS335)

## Project Overview
This project is a Class Management Web Application developed collaboratively by Group 4 as part of the CS335 course assignment.

The system is designed to help class teachers, class representatives, and **students communicate effectively by sharing announcements, learning materials, class schedules, and other important academic information.

---

## Project Objectives
- Practice collaborative software development using Git and GitHub
- Understand branching, committing, and merging
- Develop a simple multi-page web application
- Improve teamwork and version control skills

---

## Technologies Used
- HTML5
- CSS (basic styling)
- Git
- GitHub

## Instructions for Group Members
Each group member must follow the steps below carefully to ensure proper collaboration and correct Git workflow.

### 1. Clone the Repository
Clone the project repository to your local computer: by the command "git clone https://github.com/Delson869/group4-web-project-CS335.git"

Navigate to the Project Folder by command "cd group4-web-project-CS335"

Create Your Own Branch by command "git checkout -b yourname-page" Example: git checkout -b Gelas-announcements-page  NB:Do NOT work on the main branch.

Create Your Assigned Web Page
Create only one HTML file assigned to you by the group leader.
The file name must match the link provided on the landing page (index.html).
Do not edit index.html or other members’ files. Example file: announcements.html

Add Simple Page Content
Each page should include:
Page title
Brief description of its purpose
Basic HTML structure (<!DOCTYPE html>)


Commit Your Changes BY THE COMMANDS
git add yourpage.html
git commit -m "Add announcements page"

Push Your Branch to GitHub BY THE COMMAND
git push origin yourname-page

Notify the group leader


Important Rules
Do not push directly to the main branch
Work only on your assigned page
Pull updates if requested by the group leader
Follow Git best practices at all times
